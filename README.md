# ☕ Java Fundamentos

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)

Repositório de estudos em Java desenvolvido durante minha graduação em ADS. O foco está em construir uma base sólida da linguagem — entendendo não só *o que* fazer, mas *por que*.

---

## 📁 Estrutura do repositório

```
src/
├── exercicios/           → exercícios completos com lógica de negócio real
├── fundamentos/
│   ├── arrays/           → arrays e listas
│   ├── classes/
│   │   ├── associacao/   → associação entre classes (tem-um)
│   │   ├── concessionaria/
│   │   ├── pessoas/
│   │   └── sobrecarga/   → sobrecarga de métodos
│   ├── condicionais/     → if/else e switch
│   ├── lacosrepeticao/   → for, while, do-while
│   ├── modulos/          → separação em métodos/módulos
│   ├── operadores/       → operadores aritméticos e de entrada
│   └── strings/          → manipulação de Strings
└── utilitarios/          → classes de apoio reutilizáveis
```

> Cada pacote de classes possui uma subpasta `teste/` com os programas de teste correspondentes.

## 🏆 Destaques

Exercícios mais completos, que demonstram raciocínio estruturado e separação de responsabilidades:

| Arquivo | Descrição |
|---|---|
| `CadastroColaborador.java` | Menu interativo com cadastro de CLT, PJ e estagiários usando `ArrayList` |
| `PriorizaAtendimento.java` | Sistema de pontuação com bônus, penalidades e múltiplos critérios de prioridade |
| `RegistroTempoAtendimento.java` | Registro de tempos com cálculo de maior, menor e média |
| `CalculadoraImcComMetodo.java` | IMC com separação por responsabilidade em métodos |
| `CalculadoraDeDesconto.java` | Regras de desconto com múltiplas condições encadeadas |
| `RelatorioLaboratorio.java` | Relatório diário de amostras com acumuladores e formatação |

---

## 📚 Conceitos praticados

- Orientação a objetos: encapsulamento, herança, associação e sobrecarga de métodos
- Estruturas condicionais e de repetição
- Entrada e validação de dados com `Scanner`
- Coleções: `ArrayList` e arrays
- Formatação de saída com `printf`
- Organização de código em pacotes e separação de responsabilidades
- Boas práticas de nomenclatura e estrutura de projetos Java

## 🎓 Projetos de Faculdade

### Sistema de Cadastro de Colaboradores

Sistema interativo de cadastro e cálculo de salários por tipo de vínculo empregatício, desenvolvido como projeto da UC de ADS.

**Conceitos aplicados:**
- Tipos primitivos e compostos, constantes e variáveis
- Entrada e saída de dados com `Scanner` e `System.out`
- Estruturas condicionais (`if`, `else if`, `switch`)
- Estruturas de repetição (`for`, `while`, `do-while`)
- Manipulação de listas dinâmicas com `ArrayList`
- Organização do código em classes e métodos
